Profil Sekolah

PKBM Acarya Insan Cendekia berdiri dengan tujuan memberikan kesempatan belajar bagi masyarakat yang ingin menuntaskan pendidikan jenjang SD, SMP, dan SMA melalui program kesetaraan Paket A, B, dan C.

Selain itu, kami juga mengembangkan berbagai program life skill untuk meningkatkan kemandirian peserta didik.

Program Pendidikan

  • Paket A (Setara SD) – Pembelajaran dasar untuk literasi dan numerasi.
  • Paket B (Setara SMP) – Pendidikan menengah untuk penguatan karakter dan keterampilan.
  • Paket C (Setara SMA) – Pendidikan lanjutan untuk membuka peluang studi dan kerja.
